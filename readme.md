# Store‑garden — интернет‑магазин садовых принадлежностей

## Что это
Готовый сайт для продажи товаров для сада и дачи: инструменты, техника, мебель, растения, системы полива.  
Сайт сделан так, чтобы покупатель легко находил нужное, читал статьи, получал консультацию и быстро заказывал.

## Возможности
- Каталог с фильтрами (по цене, типу, сезону)
- Карточка товара с фото, описанием, видеообзором
- Корзина + онлайн‑оплата (ЮKassa, СберБанк)
- Чат с консультантом (или чат‑бот с частыми вопросами)
- Блог со статьями и видео
- Отзывы покупателей с фото
- Программа лояльности (баллы, скидки)
- Аналитика: Яндекс.Метрика, Google Analytics, пиксели соцсетей

## Технологии
- **Фронтенд**: Vue.js / React (на выбор)
- **Бэкенд**: Python (Django) или Node.js
- **База данных**: PostgreSQL
- **Поиск**: Elasticsearch для быстрых фильтров
- **Кэш**: Redis
- **Картинки**: WebP, CDN Cloudflare
- **Для мобильной версии**: адаптивный дизайн (mobile first)
